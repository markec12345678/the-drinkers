# The Drinkers

## Project Description
The Drinkers is a user-friendly application designed to help users explore, discover, and record their favorite beverages with detailed tasting notes and reviews. This application serves as a digital journal and social platform for drink enthusiasts.

## Features
- **User Profiles**: Create and manage personal profiles.
- **Drink Catalog**: Explore a wide variety of beverages categorized by type.
- **Tasting Notes**: Add, edit, and delete tasting notes for each drink.
- **Review System**: Rate and review drinks within the community.
- **Social Sharing**: Share your favorite finds and reviews with other users.

## Installation
To install The Drinkers application, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/markec12345678/the-drinkers.git
   cd the-drinkers
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up environment variables:**
   Create a `.env` file in the root of the project and provide the necessary configurations.

4. **Run the application:**
   ```bash
   npm start
   ```

## Usage
After installation, navigate to `http://localhost:3000` to access the application. You can create an account, log in, and begin exploring drinks or adding your own tasting notes.
